12.13 B 改为seen to run
解析：在主动语态中，在see, make, feel, let等动词后面的宾语补足语如有动词不定式一般都 应该省略掉，但是在被动语态的情况下，都不能省略。
14.15 C 改为how
解析：连接代词或连接副词加不定式可以作为某些动词的宾语，但是这些连接代词或连接副 词应该在句子中充当成分，而what在句子中不能充当get along with的状语，所以改用how。
16.17 D 改为to reproduce
解析：have, make, let等使役性动词在接“sb.+ 不定式”结构时，不定式应当省略，但是当 have表示“拥有”时，则不能套用此规则，本题中不定式充当power的定语，必须加to。
18.19 D 改为to blame
解析：不定式to blame, to let等作表语时，必须用主动表示被动。
20.21 D 改为to have finished
解析：根据不定式与主句谓语的时间逻辑关系，不定式的时间在主句谓语之前时应该用完 成式，因此应该用have finished。
